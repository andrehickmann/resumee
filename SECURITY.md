# Security Policy

Thanks for helping keep this project secure.

## Supported Versions

Security updates are applied to the `master` branch.

## Reporting a Vulnerability

Please report security issues by email:

- **andre@hickmann-kuschnereit.de**

Include:

- A clear description of the issue
- Steps to reproduce (if possible)
- Any relevant logs or screenshots

You will receive an acknowledgment within **7 days**. If the issue is confirmed, a fix will be prioritized and released.

## Do Not

Please do not open public issues for security vulnerabilities.
